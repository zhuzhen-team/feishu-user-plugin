# 知乎答题目标清单

**策略**：在已存在的高赞问题下回答 > 写新文章。流量比专栏文章高 3-5 倍。每篇答案 800-1500 字，结尾挂专栏链接 / GitHub repo。

**发布节奏**：知乎一次发 ≥ 3 个答案会被限流，**每天最多发 1-2 篇**，挑高质量问题。

## 目标问题搜索关键词

打开 https://www.zhihu.com/search 搜索以下 query，按"赞同数"排序找已有 ≥50 赞同 / ≥10 回答的活跃问题：

1. **"如何使用 Claude Code"** / **"Claude Code 怎么集成飞书"**
2. **"飞书机器人开发"** / **"飞书机器人 怎么开发"**
3. **"飞书 OpenAI"** / **"飞书 AI 集成"**
4. **"MCP 飞书"** / **"MCP 怎么用"**
5. **"飞书自动化"** / **"飞书 RPA"**
6. **"如何用 AI 提高工作效率"** + filter "飞书"
7. **"飞书 cookie"** / **"飞书 user_access_token"**

## 答题节奏

| Day | 操作 |
|---|---|
| Day 0 | 专栏建立 + 发首篇文章 |
| Day 1 | 想法（短动态）发版公告 |

## 监测指标

- 单篇回答首日阅读：< 1k = 问题热度不够，换；1k-10k = 正常；> 10k = 写续篇
- 收藏 / 阅读比：知乎 "收藏" 是高质量信号，> 5% 算抓住核心读者
- 进 home feed 推荐：4 小时内是否被推荐流转（看通知）
- 评论区是否出现 "请问怎么用 / 我能不能..." 这类，说明转化率正常
